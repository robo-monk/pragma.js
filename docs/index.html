About

Welcome! Pragma is a not-really-"framework", which provides a really powerful way to build UIs and apps with JavaScript. It doesn't make any assumptions about the rest of your workflow, you can use it as much as you want, or as little as you like. Its fully customizable and built to be extensible.

Its still in early development and some features are missing like the css. It depends on a couple of libraries, and jquery which can easily be removed later on.

I hope you enjoy it and create some epic shit with it.


Check out code on github , npm page, my twitter
